Announcing Lifty 9th of August 2010

Approximately 3 months ago I was selected as a Google Summer of Code student for the organization École Polytechnique Fédérale de Lausanne (EPFL). I was lucky enough to get two mentors, David Pollak and Timothy Perrett. We've been hard at work on the project since and now three months later we finally have something to show for it. Say hello to Lifty.

Lifty adds scaffolding to sbt (Simple Build Tool) and includes templates for Lift including new projects, adding snippets, models, comet actors, and other templates for Lift-base projects.

Lifty consists of two parts, Lifty-engine and Lifty.

Lifty-engine

Is a framework that helps you develop SBT processors which generate source files. All you have to do is mix in a simple trait, implement at least one Template and you're done.

Modularity has been a hight priority for us and Lifty-engine has some cool ways to construct templates. Templates can depend on other templates and inject code into templates before being processed by Scalate.

If you're interested in more information or if you want to try it out this guide will take you through the process of creating your first SBT processor using Lifty-enine here

Lifty

This is a SBT processor for generating files associated with the Lift framework. It lets you get started with Lift in less than a minute - depending on your internet connection ;) and helps you be more productive in your daily Lift activities.

Currently it has declared the following templates:

  • Snippet: A snippet
  • Mapper: Model class using Mapper
  • Comet: Comet component
  • Layout: An empty layout with a surround tag
  • Project: A Lift project with some functionality to get you started
  • Project-blank: A blank Lift project that uses SBT as it's build system
  • User: Model class implemented using Mapper. It provides basic functionality that would expect from a User

It has been built using Lifty-engine of course.

For more information about the processor check out http://github.com/Lifty/Lifty

Lifty is a plugin that adds scaffolding to SBT 0.11.x (Simple Build Tool). It's easy to teach Lifty about new scaffolding templates, you just have to give it the url to recipe and it will download the templates and you're set. See the example below for how to teach it about Lift related tempaltes

lifty learn lift https://raw.github.com/Lifty/lifty/master/lifty-recipe/lifty.json
